# プライバシーポリシー

最終更新日: 2026年3月7日

## はじめに

capsicum（以下「本アプリ」）は、Mastodon および Misskey に対応する Fediverse クライアントアプリです。本プライバシーポリシーでは、本アプリにおける情報の取り扱いについて説明します。

## 収集する情報

本アプリが端末内に保存する情報は以下の通りです。

- **アカウント認証情報** — ログイン時に取得するアクセストークンを、端末のセキュアストレージに保存します
- **アプリ設定** — 利用者が設定した各種設定値を端末内に保存します

これらの情報は端末内にのみ保存され、第三者に送信されることはありません。

## 外部との通信

本アプリは、以下のサーバーと通信を行います。

- **Mastodon / Misskey サーバー** — 利用者が接続先として選択したサーバーと通信します。通信内容はサーバーの API 仕様に準じます。各サーバーにおける情報の取り扱いについては、そのサーバーの運営者が定めるプライバシーポリシーをご確認ください
- **プッシュ通知リレーサーバー** — プッシュ通知機能を利用する場合、開発者が運営するリレーサーバー（`relay.capsicum.shrieker.net`）と通信します。リレーサーバーに送信される情報は以下の通りです
  - 端末のプッシュ通知トークン（APNs / FCM）
  - 接続先サーバーのドメイン名
  - アカウントの識別情報

リレーサーバーは、Mastodon / Misskey サーバーから受信した通知を利用者の端末に転送する目的でのみこれらの情報を使用します。アクセストークンや投稿内容など、通知の転送に不要な情報はリレーサーバーに送信されません。

## 端末機能へのアクセス

本アプリは以下の端末機能を使用する場合があります。

- **カメラ / フォトライブラリ** — 投稿にメディアを添付する目的でのみ使用します。撮影・選択されたメディアは、利用者が接続先として選択したサーバーにアップロードされます

## 分析・広告

本アプリは、分析ツールや広告 SDK を使用しません。

## 子どものプライバシー

本アプリは、13歳未満の子どもから意図的に個人情報を収集することはありません。

## 変更について

本プライバシーポリシーは変更されることがあります。変更があった場合は、本ページを更新します。
