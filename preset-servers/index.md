# プリセットサーバー

「プリセットサーバー」とは、capsicum にあらかじめ登録されている、開発者 pooza が運営・参加するコミュニティの Mastodon / Misskey サーバー群です。capsicum はもともと、これらのサーバーのユーザーに向けて開発されたアプリです。

どなたでも capsicum をお使いいただけますが、開発の優先順位はプリセットサーバーのメンバーにとっての利便性が最優先です。[プッシュ通知]((/push-notification))をはじめとする一部の機能も、プリセットサーバーのアカウントをお持ちの方を対象としています。

## プリセットサーバー一覧

| サーバー | 種別 | コミュニティ |
|----------|------|------------|
| [美食丼](https://mstdn.b-shock.org) | Mastodon | pooza の個人サーバー |
| [デルムリン丼](https://mstdn.delmulin.com) | Mastodon | ダイの大冒険 |
| [キュアスタ！](https://precure.ml) | Mastodon | プリキュア |
| [ダイスキー](https://misskey.delmulin.com) | Misskey | ダイの大冒険 |
| [きゅあすきー](https://mk.precure.fun) | Misskey | プリキュア |

きゅあすきーのみ、運営者は開発者 pooza ではありません。キュアスタ！の運営にも携わり、capsicum・[モロヘイヤ](https://github.com/pooza/mulukhiya-toot-proxy)にも日頃から意見を寄せてくれる、共同管理人に近い立場の方が運営しています。

## プリセットサーバーに登録するには

現在プリセットサーバーにアカウントをお持ちでない方も、サーバーに登録することで、[プッシュ通知]((/push-notification))をはじめとするプリセットサーバー向けの機能を利用できるようになります。

### プリキュアのファンの方

**キュアスタ！**（Mastodon）または **きゅあすきー**（Misskey）に登録できます。

- キュアスタ！は承認制と招待制を併用しています。登録時にプリキュアファンであることが伝わる意気込みを書いていただければ、承認されるでしょう
- きゅあすきーは招待制です。既にメンバーである方（開発者の pooza を含む）に招待を依頼してください

### ダイの大冒険のファンの方

**デルムリン丼**（Mastodon）または **ダイスキー**（Misskey）に登録できます。

- デルムリン丼は承認制と招待制を併用しています。登録時にダイの大冒険のファンであることが伝わる意気込みを書いていただければ、承認されるでしょう
- ダイスキーは招待制です。既にメンバーである方（開発者の pooza を含む）に招待を依頼してください

### その他の方

開発者 pooza の個人的な知り合いであれば、**美食丼**（Mastodon）に登録できます。capsicum を利用したい旨の意気込みを書いていただければ、多くの場合承認されるでしょう。

## プッシュ通知について

プリセットサーバーのアカウントをお持ちの方には、Mastodon / Misskey サーバーからのプッシュ通知を無償で提供します。詳しくは[プッシュ通知について](/push-notification)をご覧ください。
